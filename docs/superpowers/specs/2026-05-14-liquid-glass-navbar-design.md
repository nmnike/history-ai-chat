# Дизайн верхней панели Liquid Glass

## Цель

Заменить текущую чёрную Bootstrap-панель на современную сдержанную Liquid Glass шапку без сильного изменения геометрии приложения.

## Решение

Использовать full-width navbar с отдельным классом `app-navbar`. Панель остаётся на прежнем месте и сохраняет текущую структуру Bootstrap collapse, но получает стеклянный визуальный слой:

- полупрозрачный тёмный градиент вместо плоского `bg-dark`;
- `backdrop-filter: blur(...) saturate(...)` для поддерживаемых браузеров;
- тонкая светлая граница снизу;
- мягкая тень и inset-highlight;
- fallback через обычный тёмный фон, если blur не поддерживается.

## Изменения

- В `src/viewer/templates/base.html` заменить классы `navbar-dark bg-dark` на `navbar-dark app-navbar`.
- В `src/viewer/static/css/theme.css` добавить стили `.app-navbar` рядом с бренд-стилями или Bootstrap overrides.
- Обновить стиль кнопки `#theme-toggle` внутри `.app-navbar`, чтобы он выглядел как стеклянная outline-кнопка.
- Сохранить текущую высоту, sticky-поведение Bootstrap и мобильный collapse без переработки layout.

## Визуальные ограничения

- Эффект должен быть сдержанным: не floating pill и не сильная прозрачность.
- Текст `History AI Chat`, версия и кнопка темы должны оставаться читаемыми на светлой и тёмной темах.
- Новая SVG-иконка приложения должна визуально сочетаться с шапкой.

## Проверка

- Запустить тесты проекта.
- Запустить приложение и проверить в браузере:
  - верхняя панель больше не выглядит плоской чёрной полосой;
  - виден blur/glass эффект и тонкая граница;
  - navbar корректно выглядит на dashboard, project и conversation страницах;
  - в консоли нет новых ошибок.
