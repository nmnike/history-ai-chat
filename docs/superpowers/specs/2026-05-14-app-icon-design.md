# Дизайн иконки приложения

## Цель

Заменить текущую составную иконку рядом с надписью `History AI Chat` и favicon браузера на единый material-style знак в образе `AI + чат`.

## Решение

Создать один SVG-ассет `src/viewer/static/app-icon.svg` и использовать его в двух местах:

- favicon через `<link rel="icon" type="image/svg+xml" href="/static/app-icon.svg">`;
- бренд в navbar через `<img class="brand-icon" src="/static/app-icon.svg" alt="">` рядом с текстом `History AI Chat`.

Визуально иконка: скруглённый квадрат с материал-дизайн ощущением, сине-фиолетовый градиент, белый пузырь чата и небольшой AI/sparkle-акцент. Она должна быть читаемой в маленьком размере вкладки браузера и выглядеть аккуратно в тёмной navbar.

## Изменения

- Добавить новый SVG-файл иконки.
- Обновить favicon в `src/viewer/templates/base.html`.
- Заменить две Bootstrap Icons в бренде на один `<img>`.
- Упростить CSS `.brand-icon` до размеров, object-fit и лёгкой тени.
- Оставить старый `favicon.svg` без удаления, чтобы не расширять scope.

## Проверка

- Запустить тесты проекта.
- Запустить приложение и проверить в браузере:
  - иконка отображается рядом с `History AI Chat`;
  - favicon во вкладке обновился;
  - нет ошибок в консоли браузера.
